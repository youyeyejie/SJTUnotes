# 第二章 新民主主义革命理论

## 新民主主义革命理论形成的依据

- 认清中国的国情是认清一切革命问题的基本的根据，是解决中国革命问题的基本前提。

- 近代中国国情：

	- 性质：**半殖民地半封建社会**

	- 主要矛盾：

		- **帝国主义和中华民族的矛盾**（最主要矛盾）

		- 封建主义和人民大众的矛盾

	- 主要矛盾决定根本任务：推翻三座大山

		- 帝国主义

		- 封建主义

		- 官僚资本主义

	- 革命性质：**资产阶级民主革命**（旧 + 新）

		- 革命性质由革命对象决定，与领导阶级无关：

			- 旧民主主义革命：推翻帝国主义和封建主义的统治

			- 新民主主义革命：推翻帝国主义、封建主义和官僚资本主义的统治

		- 新民主主义革命不是无产阶级社会主义革命，因为革命主要对象并不是资产阶级，主要矛盾不是资本主义和无产阶级的矛盾

- 近代中国革命的时代特征

	- 旧民主主义革命：鸦片战争到辛亥革命期间，资产阶级民主革命

	- 新民主主义革命：十月革命到第二次世界大战结束期间，仍是资产阶级民主革命，但成为 **世界无产阶级社会主义革命的一部分**

		- 以五四运动为标志，中国无产阶级开始作为独立的政治力量登上历史舞台，成为革命的领导力量

		- 旧民主主义革命与新民主主义革命的区别：革命领导力量不同、指导思想不同、革命对象不同、革命前途不同

			- 区分的根本标志：**无产阶级领导权**

		- 新民主主义革命与无产阶级社会主义革命的区别：新民主主义只推翻帝国主义、封建主义和官僚资本主义的反动统治，而不破坏参与反帝反封建的资本主义成分。

- **中国革命分两步走**：

	1. 反帝反封建的新民主主义革命

	2. 社会主义革命

- 新民主主义革命理论的实践基础

	- 旧民主主义革命的失败、近代中国革命形势的发展和世界形势的新变化，呼唤新的革命理论

	- **新民主主义的革命实践** 是新民主主义理论得以形成的实践基础

## 新民主主义革命的总路线和基本纲领
### 新民主主义革命的总路线

- 总路线：**无产阶级领导的，人民大众的，反对帝国主义、封建主义和官僚资本主义的革命。**

	- 首次提出：1939 年，毛泽东《中国革命和中国共产党》

	- 完整表述：1948 年，毛泽东《在晋绥干部会议上的讲话》

- **领导力量**：无产阶级

	- 无产阶级的领导权是中国革命的 **中心问题**，也是新民主主义革命理论的 **核心问题**。

	- 区分新旧两种不同范畴的民主主义革命的根本标志：**无产阶级领导权**

	- 中国无产阶级的优点与特点：

		- 一般无产阶级的基本优点：与先进的生产方式相联系、没有私人占有的生产资料、富于组织纪律性等

		- （决心强）一是它从诞生之日起，就身受外国资本主义、本国封建势力和资产阶级的三重压迫

		- （集中性）二是它集中分布在少数大城市和大企业，有利于组织和团结，有利于革命思想的传播和强大革命力量的形成

		- （同盟性）三是它的成员中大部分出身于破产农民，和农民有着天然的联系，便于和农民结成亲密的联盟

- **革命动力**：人民大众

	- 工人阶级：**中国革命的最基本动力**

	- 农民阶级：**中国革命主力军**

	- 城市小资产阶级：无产阶级的可靠同盟者

	- 民族资产阶级

- **革命对象**：**帝国主义**（首要对象）、封建主义、官僚资本主义（三座大山）

	- 分清敌友，这是革命的首要问题。

	- 中国革命，就是对外推翻帝国主义压迫的 **民族革命**（最主要任务），对内推翻封建地主压迫的 **民主革命**。

	- 不同时期主要矛盾与革命对象的变化

		- 大革命时期：帝国主义支持的北洋军阀

		- 1927 土地革命时期：国民党新军阀

		- 1931 抗日战争：日本帝国主义

		- 1945 解放战争：美帝国主义支持的国民党反动派

- 性质和前途

	- 性质：**资产阶级民主主义革命**

	- 前途：**社会主义**

	- 与旧民主主义革命相比有特点与新内容：是 **世界无产阶级社会主义革命的一部分**

		- 根本区别：革命领导力量是中国无产阶级及其先锋队——中国共产党

		- 指导思想：马列主义

		- 前途：社会主义（而非资本主义）

	- 与社会主义革命的关系：

		- 区别：性质不同

			- 新民主主义革命要建立的是无产阶级领导的各革命阶级的联合专政，而不是无产阶级专政。

			- 社会主义革命是无产阶级性质的革命，目标是消灭资产阶级剥削和改造小生产的私有制。

		- 联系：互相联系、紧密衔接，中间不容横插一个资产阶级专政。

			- **两步走**：第一步是民主主义革命，第二步是社会主义革命

			- 民主主义革命是社会主义革命的必要准备，社会主义革命是民主主义革命的必然趋势

### 新民主主义的基本纲领

- 基本纲领：**政治纲领、经济纲领、文化纲领**

	- 新民主主义政治经济文化的阐述：1940 年，毛泽东《新民主主义论》

	- 纲领的具体阐述与形成：1945 年，毛泽东，中共七大政治报告《论联合政府》

- **政治纲领**：推翻帝国主义和封建主义的统治，建立一个无产阶级领导的、以工农联盟为基础的、各革命阶级联合专政的新民主主义的共和国。

- **经济纲领**：没收封建地主阶级的土地归农民所有（主要内容），没收官僚资产阶级的垄断资本归新民主主义的国家所有，保护民族工商业。

- **文化纲领**：无产阶级领导的人民大众的反帝反封建的文化，即民族的科学的大众的文化。

## 新民主主义革命的道路和基本经验
### 新民主主义革命的道路
革命道路：**农村包围城市、武装夺取政权**

- 形成：

	- 提出：1927 年大革命失败后，毛泽东在在创建和领导井冈山革命根据地的革命斗争实践中，写下《中国的红色政权为什么能够存在?》《井冈山的斗争》《星星之火，可以燎原》等文章中提出了“工农武装割据”思想，初步形成了农村包围城市的革命道路理论。

	- **确立**：1938 年，毛泽东在中共六届六中全会确立了经过长期武装斗争，先占乡村，后取城市，最后夺取全国胜利的革命道路。

- 必然性：时代特点和具体国情决定

	- 半殖民地半封建，内无民主外无独立，革命的主要形式只能是武装斗争。

	- 中国是农业大国，农民绝大多数，解决农民的土地问题才能使革命战争获得广大农民的支持和参加。

- 根本：处理好土地革命、武装斗争、农村革命根据地建设三者之间的关系（工农武装割据）

	- **土地革命**：民主革命的基本内容

	- **武装斗争**：中国革命的主要形式、根据地建设和土地革命强有力保证

	- **农村革命根据地**：中国革命的战略阵地、武装斗争和开展土地革命的依托

### 新民主主义革命的三大法宝

- 三大法宝：**统一战线、武装斗争、党的建设**

	- 提出：毛泽东《〈共产党人〉发刊词》

- **统一战线**：**无产阶级政党策略思想的重要内容**

	- 必要性：

		- 由中国半殖民地半封建社会的阶级状况所决定

		- 由中国革命的长期性、残酷性及其发展的不平衡性所决定

	- 可能性：半殖民地半封建的中国社会交织在一起的诸多矛盾

	- 历程：第一次国共合作的统一战线、工农民主统一战线、抗日民族统一战线、人民民主统一战线

	- 内容：**两个联盟**

		- **工农联盟**：基础，是基本的、主要的

		- **与民族资产阶级的联盟**：不可缺少的、非基本的、辅助的

	- 方法论：

		- （对革命主力）建立巩固的工农联盟

		- （对资产阶级群体）正确对待资产阶级，尤其是民族资产阶级：对资产阶级既联合又斗争

		- （对个人）采取区别对待的方针：发展进步势力、争取中间势力、孤立顽固势力

		- （对党）坚持独立自主的原则：保持党在政治上、组织上和思想上的独立性。

- **武装斗争**：**中国革命的特点和优点之一、中国革命的主要斗争形式**

	- 建设经验：

		- **坚持党对军队的绝对领导**：建设新型人民军队的根本原则，保持人民军队无产阶级性质和建军宗旨的根本前提，毛泽东建军思想的核心

		- 建设以 **全心全意为人民服务** 为唯一宗旨的人民军队：建设新型人民军队的基本前提， 也是人民军队一切行动的根本准则和一切工作的出发点与归宿。

		- 开展革命的政治工作

		- 坚持正确的战略战术原则

- **党的建设**：**必须不断加强党的思想建设、组织建设和作风建设**

	- 必要性：

		- 农民和小资产阶级出身的党员占多数

		- 党内无产阶级思想和非无产阶级思想之间的矛盾成为党内思想上的主要矛盾

	- 内容：三大建设

		- **思想建设**：党的建设的 **首位**，关键是要以无产阶级思想克服和改造各种非无产阶级思想。

		- **组织建设**：根本的是要贯彻民主集中制这一根本组织原则。

		- **作风建设**：形成理论联系实际、密切联系群众、批评与自我批评相结合的三大优良作风。

		- 联系党的政治路线加强党的建设

### 新民主主义革命理论的意义

- 理论意义：

	- 新民主主义革命理论，揭示了近代中国革命发展的客观规律，解决了在一个以农民为主体的、落后的半殖民地半封建的东方大国进行革命的一系列理论问题。科学地回答了近代中国革命向何处去的问题，正确地解决了中国革命的发展阶段问题，极大地丰富了马克思主义的理论宝库。

	- 新民主主义革命理论是马克思主义中国化时代化的重要理论成果，开辟了马克思主义中国化时代化的发展道路。是对中国革命实践经验的概括和总结，是中国共产党集体智慧的结晶。

- 实践意义：

	- 指导党团结带领人民找到了一条正确革命道路，完成了新民主主义革命，建立了中华人民共和国，开创了中国历史的新纪元。

	- 中国新民主主义革命的伟大胜利，是 20 世纪继俄国十月革命之后改变世界面貌的伟大历史事件。极大改变了世界政治格局，鼓舞了全世界被压迫民族和被压迫人民争取解放的斗争，极大地增强了他们反对帝国主义斗争的信心，增强了世界人民争取和平的力量。
