???+ info "我国社会各阶段性质"

	- **近代中国**（1840-1949 年）：半殖民地半封建社会
	- **社会主义过渡时期**（1949-1956 年）：新民主主义社会
	- **社会主义改造完成后**（1956 年至今）：社会主义社会（社会主义初级阶段）

???+ info "我国社会各阶段主要矛盾"

	- **近代中国**（1840-1949 年）：帝国主义和中华民族的矛盾、封建主义和人民大众的矛盾
	- **社会主义过渡**（国民经济恢复）时期（1949-1952 年）：人民大众同帝国主义、封建主义、和国民党反动派残余势力之间的矛盾
	- **社会主义过渡**（一化三改）时期（1952-1956 年）：工人阶级和资产阶级两个阶级的矛盾、社会主义和资本主义两条道路的矛盾
	- **社会主义建设时期**（1956-1978 年）：人民对于建立先进的工业国的要求同落后的农业国的现实之间的矛盾、人民对于经济文化迅速发展的同当前经济文化不能满足人民需要的状况之间的矛盾
	- **改革开放新时期**（1978-2017 年）：人民日益增长的物质文化需要同落后的社会生产之间的矛盾
	- **中国特色社会主义新时代**（2017 年至今）：人民日益增长的美好生活需要和不平衡不充分的发展之间的矛盾

	> 基本矛盾：生产力与生产关系、经济基础与上层建筑之间的矛盾

		- 理论成果的关系：一脉相承又与时俱进
	- **[毛泽东思想](第一章-毛泽东思想及其历史地位.md)**
		- **活的灵魂**
			- 实事求是（精髓）：一切从实际出发，理论联系实际，在实践中检验和发展真理
			- 群众路线：一切为了群众、一切依靠群众，从群众中来、到群众中去，把党的正确主张变为群众的自觉行动
			- 独立自主：坚持独立思考，走自己的路
		- **主要内容**
			- **[新民主主义革命理论](第二章-新民主主义革命理论.md)**
				- **主要内容**
					- 总路线：无产阶级领导的，人民大众的，反对帝国主义、封建主义和官僚资本主义的革命
					- 革命性质：资产阶级民主主义革命
					- 中国革命两步走：反帝反封建的新民主主义革命、社会主义革命
					- 基本纲领
						- 政治纲领：推翻帝国主义与封建主义的通知，建立无产阶级领导的、以工农联盟为基础的、各革命阶级联合专政的新民主主义共和国
						- 经济纲领：没收封建地主阶级的土地归农民所有，没收官僚资产阶级的垄断资本归国家所有，保护民族工商业
						- 文化纲领：无产阶级领导的、人民大众的、反帝反封建的文化，民族的、科学的、大众的文化
					- 道路：农村包围城市、武装夺取政权
					- 三大法宝：统一战线、武装斗争、党的建设
				- **意义**
					- （理论+马）：揭示了近代中国革命发展的客观规律，科学地回答了近代中国革命向何处去的问题，是马克思主义中国化时代化的重要理论成果，丰富和发展了马克思主义，是中国共产党集体智慧的结晶。
					- （实践+世界）：指导党团结带领人民找到了一条正确革命道路，完成了新民主主义革命，建立了中华人民共和国，开创了中国历史的新纪元，改变了世界面貌，增强了世界人民争取和平的力量。
			- **[社会主义革命理论](第三章-社会主义改造理论.md)**
				- **主要内容**
					- 新民主主义社会的性质：新民主主义到社会主义的过渡时期
					- 党在过渡时期的总路线：一化三改——要在一个相当长的时期内，逐步实现国家的社会主义工业化，并逐步实现国家对农业，对手工业和对资本主义工商业的社会主义改造
					- 社会主义改造道路
						- 农业、手工业：走互助合作道路；遵循自愿互利、典型示范、国家帮助的原则；分析农村阶级阶层状况，指定正确的阶级政策；坚持积极领导、稳步前进的方针，采取循序渐进的步骤
						- 资本主义工商业：以和平赎买的方式改造；采取从低级到高级的国家资本主义的过渡方式；将资本主义工商业者改造为社会主义劳动者
				- **确立社会主义基本制度的意义**
					- （**地位**）：是中国历史上最深刻最伟大的社会变革，为当代中国一切发展进步奠定了制度基础，也为中国特色社会主义制度的创新和发展提供了重要前提。
					- （实践+世界）：确立了社会主义基本制度，促进生产力发展，显示了社会主义制度的优越性，使人民当家做主、阶级关系发生根本变化，改变世界政治经济格局。
					- （理论+马）：是对中国如何进行社会主义革命问题的系统回答和正确解决，丰富和发展了马克思主义的科学社会主义理论。
			- **[社会主义建设理论](第四章-社会主义建设道路初步探索.md)**
				- **主要内容**
					- 《论十大关系》确定的基本方针：调动一切积极因素为社会主义事业服务
					- 正确认识和处理社会主义社会矛盾的思想：基本矛盾是生产力与生产关系、经济基础与上层建筑之间的矛盾；主要矛盾是人民对于建立先进的工业国的要求与落后的农业国的显示之间的矛盾、人民对于经济文化迅速发展的需要同当前经济文化不能满足人民需要的状况之间的矛盾
					- 走中国工业化道路的思想：两条腿走路
					- “四个现代化”战略目标：农业现代化、工业现代化、国防现代化、科学技术现代化
					- 党的建设理论：三大作风——理论联系实际、密切联系群众、批评与自我批评相结合
				- **意义**
					- （制度）巩固和发展了我国的社会主义制度。
					- （中特）为开创中国特色社会主义提供了宝贵经验、理论准备、物质基础。
					- （马）丰富了马克思主义科学社会主义的理论和实践。
		- **历史地位**
			- （马）马克思主义中国化时代化的第一个重大理论成果。
			- （指南）中国革命和建设的科学指南。
			- （财富）中国共产党和中国人民宝贵的精神财富。
	- **[邓小平理论](第六章-邓小平理论.md)**
		- **核心思想**：“走自己的道路，建设有中国特色的社会主义”
		- **社会主义的本质**：解放生产力，发展生产力，消灭剥削，消除两极分化，最终达到共同富裕
		- **主要内容**
			- 社会主义初级阶段理论
				- 基本国情：我国处于并将长期处于社会主义初级阶段
				- 党的基本路线：领导和团结全国各族人民，以经济建设为中心，坚持四项基本原则，坚持改革开放，自力更生，艰苦创业，为把我国建设成为富强、民主、文明的社会主义现代化国家而奋斗。
				- 四项基本原则：坚持社会主义道路、坚持人民民主专政、坚持党的领导、坚持马列毛思想
			- 社会主义根本任务和发展战略理论
				- 根本任务：发展社会生产力
				- 党和国家的工作重点：经济建设
				- 三步走发展战略：温饱、小康、基本实现现代化
			- 改革开放理论
				- 改革是一场深刻的社会变革，是中国的第二次革命，是实现中国现代化的必由之路，是社会主义制度的自我完善和发展
				- 改革是社会主义社会发展的第一动力
				- 评判挨个是非得失的标准：是否有利于生产力发展、综合国力增强、人民生活水平提高
				- 对外开放是基本国策
			- 社会主义市场经济理论
				- 计划经济和市场经济不是划分社会制度的标志
				- 计划和市场都是经济手段
				- 市场经济作为资源配置的手段本身不具有制度属性
			- “两手抓，两手都要硬”
				- 物质文明与精神文明
				- 建设与法制
				- 改革开放与惩治腐败
			- 和平与发展是当今世界的两大主题
		- **历史地位**
			- （马）马克思列宁主义、毛泽东思想的继承和发展
			- （中特）中国特色社会主义理论体系的开篇之作
			- （指南）改革开放和社会主义现代化建设的科学指南
	- **[三个代表重要思想](第七章-三个代表重要思想.md)**
		- **核心观点**
			- 党必须始终代表中国先进社会生产力的发展要求
			- 党必须始终代表中国先进文化的前进方向
			- 党必须始终代表中国最广大人民的根本利益
		- **主要内容**
			- 发展是党执政兴国的第一要务
			- 建立社会主义市场经济体制
			- 引进来和走出去并重的对外开放战略
		- **历史地位**
			- （中特）中国特色社会主义理论体系的丰富发展
			- （武器）加强和改进党的建设、推进中国特色社会主义事业的理论武器
	- **[科学发展观](第八章-科学发展观.md)**
		- **科学内涵**
			- 第一要义：推动经济社会发展
			- 核心立场：以人为本
			- 基本要求：全面协调可持续发展
			- 根本方法：统筹兼顾
		- **主要内容**
			- （经济-富强）加快转变经济发展方式
			- （政治-民主）发展社会主义民主政治
			- （文化-文明）推进社会主义文化强国建设
			- （社会-和谐）构建社会主义和谐社会
			- （生态-美丽）推进生态文明建设
			- 全面提高党的建设科学化水平
		- **历史地位**
			- （中特）中国特色社会主义理论体系在新阶段的持续发展
			- （指针）全面建成小康社会、加快推进社会主义现代化的根本指针
