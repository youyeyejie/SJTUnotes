# 第五章 信息化装备

## 信息化装备概述

- 信息化装备对现代战争的影响  
	- 侦察立体化、反应快速化、打击精确化、防护综合化、控制智能化。  
- 信息化作战平台发展趋势
	- 隐形化、多功能化、无人化。
- 信息化弹药发展趋势
	- 灵巧型、智能性。

## 信息化作战平台

- 坦克
	- 具有强大直射火力、高度越野机动性和装甲防护力的履带式装甲战斗车辆（如美国 M-1“阿布拉姆斯”、俄罗斯 T-90、中国 99 式）。
- 海上作战平台
	- 在海洋范围活动的舰艇，如航空母舰、潜艇。
- 信息化空中作战平台
	- 战斗机、轰炸机、预警机、空中加油机、直升机、军用运输机。

## 综合电子信息系统

- 情报侦察系统
	- 组成：
		- 战略、战役、战术情报侦察系统
		- 谍报人员和人民群众情报侦察系统
		- 电子战情报侦察系统
	- 分类：航天、航空、海上及水下、地面侦察系统。
- 卫星导航系统
	- 美国 GPS、俄罗斯 GLONASS、欧洲伽利略、中国北斗。

## 信息化杀伤武器

- 精准制导武器
	- 采用精确制导技术，直接命中率 50%以上的武器
- **精准制导方式**：
	- 自主制导（惯性、地形匹配、数字景象匹配、卫星定位）
	- 寻的制导（主动、半主动、被动）、遥控制导（指令、波束）
	- 复合制导
- 精确制导弹药：
	- 末制导弹药
	- 末敏弹药
- 精确制导特点：
	- 命中精度高、作战效能高、远程突防能力强、作战效费比高。
- 核武器
	- 原子弹（裂变）、氢弹（聚变）、中子弹（利用聚变效应）。
- 新概念武器
	- 激光武器：烧蚀效应、激波效应、辐射效应。
	- 粒子束武器、微波武器、动能武器、气象武器、非致命武器、基因武器、纳米武器。
